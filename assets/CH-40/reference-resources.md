## 官方设计系统文档

### Material Design（Google）

| 信息 | 内容 |
|------|------|
| **官网** | https://m3.material.io |
| **特点** | Google 的设计系统，基于「Material is the Metaphor」理念，强调物理世界的视觉隐喻 |
| **适用场景** | Android 应用、跨平台 Web 应用 |
| **核心原则** | Material is the Metaphor、Bold Graphic Intentional、Motion Provides Meaning |

### Apple Human Interface Guidelines（Apple HIG）

| 信息 | 内容 |
|------|------|
| **官网** | https://developer.apple.com/design/ |
| **特点** | Apple 的设计指南，强调平台原生体验、清晰性、依从性 |
| **适用场景** | iOS、macOS、watchOS、tvOS 应用 |
| **核心原则** | 清晰性（Clarity）、依从性（Deference）、深度（Depth） |

### Ant Design（阿里巴巴）

| 信息 | 内容 |
|------|------|
| **官网** | https://ant.design |
| **特点** | 阿里巴巴的企业级设计系统，专注中后台应用，中文生态 |
| **适用场景** | 企业级 Web 应用、React 项目 |
| **核心原则** | 自然（Natural）、确定性（Certain）、意义感（Meaning）、生长性（Growth） |

## 小团队起步工具

### 可直接使用的设计系统

| 系统 | 说明 | 适用场景 |
|------|------|---------|
| **Tailwind UI** | Tailwind CSS 官方组件库 | 快速搭建 Web 应用 |
| **MUI (Material-UI)** | Material Design 的 React 实现 | React 项目 |
| **Chakra UI** | 简洁、模块化的 React 组件库 | React 小团队 |
| **Radix UI** | 无样式组件库，可自定义 | 需要高度定制 |

### 建议

> 「不要重复发明轮子」：小团队可以先使用成熟的设计系统（如 MUI、Tailwind UI），在积累足够的定制需求后，再考虑搭建自己的设计系统。
