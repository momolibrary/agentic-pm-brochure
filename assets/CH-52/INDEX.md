## 类比素材汇总

### 核心类比

1. **登山类比**（analogy-journey-map.md）
   - 学习路径像登山：从山脚（不懂技术）→ 山腰（能对话）→ 山顶（能交付）
   - 每个阶段有不同的风景和挑战
   - 不是终点而是起点：登顶后发现还有更高的山峰

2. **盖楼类比**（analogy-journey-map.md）
   - 能力建设像盖楼：地基（认知域）→ 框架（方法域）→ 装修（实践域）
   - 地基不稳，楼盖不高
   - AI是放大器：基础认知越扎实，放大效果越好

3. **指挥家类比**（analogy-super-individual.md）
   - Human = 指挥家（判断力）
   - Agent = 乐团（执行力）
   - 判断力放大器，不是人被替代
