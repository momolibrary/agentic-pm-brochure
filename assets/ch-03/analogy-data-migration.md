## 核心类比：搬家不只是搬东西

### 主类比

**数据迁移 = 搬家**

很多人以为搬家就是「把东西从 A 搬到 B」，但实际上：

> 「搬家不只是搬东西，还有：地址变更、水电燃气过户、邮件转投、物业交接、钥匙置换、家具拆装、物品分类打包、路线规划、搬运保险、临时存储、入住检查、旧房清理……」

同样，**数据迁移不只是拷贝数据**，还有：

---

### 类比展开表

| 搬家步骤 | 数据迁移对应 | 解释 |
|---------|-------------|------|
| **物品盘点** | 数据发现与评估 | 先知道「有什么要搬」 |
| **分类打包** | 数据分类与清洗 | 分类整理，丢弃无用数据 |
| **家具拆装** | 数据格式转换 | 旧系统格式 → 新系统格式 |
| **路线规划** | 迁移策略设计 | 批量迁移还是分批迁移？ |
| **搬运保险** | 数据备份与回滚 | 出问题能恢复 |
| **地址变更** | 接口地址更新 | API 地址、连接串更新 |
| **水电过户** | 权限与认证迁移 | 用户权限、访问控制迁移 |
| **邮件转投** | 数据流与通知迁移 | 数据管道、告警配置迁移 |
| **物业交接** | 系统依赖交接 | 上游/下游系统对接调整 |
| **钥匙置换** | 密钥与凭证迁移 | 加密密钥、访问凭证迁移 |
| **临时存储** | 中间存储/缓存 | 迁移过程中的临时数据存储 |
| **入住检查** | 数据验证与验收 | 检查数据完整性、准确性 |
| **旧房清理** | 旧系统清理 | 数据残留、系统下线 |

## 关键风险类比

### 风险 1：物品丢失 → 数据丢失

**搬家场景**：
- 箱子丢了一个，里面的东西全没了
- 运输过程中物品损坏

**数据迁移场景**：
- 迁移脚本遗漏某些表/字段
- 转换过程中数据格式错误导致丢失
- **Gartner 报告：83% 数据迁移项目未达预期**

### 风险 2：地址没改到位 → 接口配置遗漏

**搬家场景**：
- 通知了银行新地址，但忘了通知信用卡中心
- 水电过户了，但燃气没过户

**数据迁移场景**：
- 新数据库地址通知了 A 系统，忘了通知 B 系统
- API 地址更新了，但缓存层还在用旧地址
- **系统间依赖关系复杂，遗漏导致服务中断**

### 风险 3：新旧钥匙冲突 → 权限冲突

**搬家场景**：
- 新房钥匙还没拿到，旧房钥匙已经失效
- 多人钥匙配置不统一

**数据迁移场景**：
- 新系统权限还没配置好，旧系统已下线
- 多系统权限不一致，导致「能看旧数据，不能看新数据」

### 风险 4：时间紧迫强行搬迁 → 业务中断

**搬家场景**：
- 「明天必须搬」，结果很多东西没整理好
- 搬进新房发现水电还没开通

**数据迁移场景**：
- 「周五必须上线」，数据还没验证
- 新系统上线发现接口地址没更新，服务直接中断
- **McKinsey：70% 数字化转型未达预期，数据迁移是主要瓶颈**

## 数据迁移的「隐藏步骤」

### 产品经理容易忽略的步骤

| 步骤 | 为什么容易被忽略 | 实际影响 |
|------|----------------|---------|
| **数据清洗** | 「数据搬过去就行」 | 新系统收到脏数据，功能异常 |
| **格式转换** | 「格式应该兼容」 | 字段类型不匹配，插入失败 |
| **历史数据** | 「只要近期数据」 | 需要历史数据时发现没了 |
| **回滚方案** | 「不会出问题」 | 出问题后无法恢复 |
| **并行运行** | 「直接切换」 | 切换失败，业务中断 |
| **用户通知** | 「技术的事」 | 用户不知道系统变了，找不到数据 |

## 企业级迁移最佳实践（供参考）

| 阶段 | 关键动作 | 风险控制 |
|------|---------|---------|
| **准备** | 数据盘点、评估、清洗 | 数据质量检查 |
| **设计** | 迁移策略、回滚方案 | 制定时间窗口 |
| **执行** | 分批迁移、并行运行 | 实时监控、日志 |
| **验收** | 数据验证、用户测试 | 功能回归测试 |
| **清理** | 旧系统下线 | 数据残留清理 |

## 可引用的关键句

> 「数据迁移不是把数据从 A 搬到 B，而是把数据从 A **转换、清洗、验证、对接、备份**之后搬到 B。」

> 「83% 的数据迁移项目未达预期，问题不在技术，在于低估了复杂度。」

> 「你以为迁移是搬家，实际是搬家 + 地址变更 + 水电过户 + 邮件转投 + ……」

## 其他类比选项

### 类比 2：装修不只是刷墙

> 「装修不只是刷墙，还有水电改造、管线重铺、家具更换、门窗调整、通风换气、安全检查……」

**适用场景**：系统升级改造（而非全新迁移）

### 类比 3：换手机不只是转移数据

> 「换手机不只是转移照片，还有：APP 重装、账号重登、通知重配、支付绑定更新、蓝牙设备重连……」

**适用场景**：终端/客户端迁移场景
