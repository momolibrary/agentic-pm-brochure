# 类比：开门成本 → 开餐厅不只是买食材

> 素材类型：类比素材

## 类比核心

**开餐厅不只是买食材。** 租金、装修、证件、设备都是开门就要花的钱——还没卖出一道菜，这些钱已经花出去了。

同样，**IT 系统不只是写代码。** 服务器、域名、证书、监控、备份都是开门就要花的钱——还没写出一行代码，这些钱已经花出去了。

---

## 详细类比表

| 餐厅开门成本 | IT 系统开门成本 | 说明 |
|--------------|-----------------|------|
| **店铺租金** | **云服务器费用** | 要有地方"开店"，要租场地/租服务器 |
| **装修费用** | **域名、DNS 配置** | 店面要装修让顾客能找到，系统要域名让用户能访问 |
| **营业执照** | **SSL 证书** | 开店要证件证明正规，HTTPS 证书证明网站安全可信 |
| **厨房设备** | **数据库服务** | 要有锅灶才能做菜，要有数据库才能存数据 |
| **收银系统** | **监控告警系统** | 要知道店里经营情况，要监控系统运行状态 |
| **消防设备** | **备份系统** | 安全设备平时不用但必须有，备份系统平时不看但必须配 |
| **员工培训** | **日志系统** | 员工要知道怎么记录经营情况，系统要知道怎么记录运行日志 |

## 类比叙述文本

### 标准叙述版本

老李说："开餐厅不只是买食材。你想想，租金要钱、装修要钱、营业执照要钱、厨房设备要钱、消防设备要钱……还没卖出一道菜，这些钱已经花出去了。

IT 系统也一样。服务器要钱、域名要钱、证书要钱、监控要钱、备份要钱……还没写一行代码，这些钱已经花出去了。这叫开门成本——还没开门营业就要花的钱。"

### 精简叙述版本

"开餐厅不只是买食材。租金、装修、证件、设备都是开门就要花的钱。IT 系统也一样——服务器、域名、证书、监控、备份，还没写代码就已经花钱了。"

## 类比优势分析

| 优势 | 说明 |
|------|------|
| **直观易懂** | 餐厅是 PM 能理解的业务场景 |
| **结构相似** | 餐厅开门成本结构清晰，与 IT 系统高度对应 |
| **隐性成本凸显** | 餐厅隐性成本（租金、装修）容易被忽视，类比能凸显这一点 |
| **商业视角** | 从商业成本视角解释，PM 更容易理解"贵"的原因 |
