# VS Code vs JetBrains vs Cursor 对比

> 素材来源：网络搜索整理
> 用途：A8-01 知识点工具对比

---

## 三者定位

| IDE | 定位 | 目标用户 |
|-----|------|---------|
| **VS Code** | 免费、轻量、插件丰富 | 全语言开发者、学生、初创团队 |
| **JetBrains** | 专业、深度、收费 | Java/Kotlin 企业开发者、大型项目 |
| **Cursor** | AI 原生、智能辅助 | AI 协作开发者、效率优先者 |

---

## VS Code（Visual Studio Code）

**微软出品，免费开源**

### 核心特点
- **轻量启动快**：几秒打开，不像传统 IDE 要等半分钟
- **插件生态庞大**：几乎任何语言都有插件支持
- **免费**：零成本，适合个人和小团队
- **跨平台**：Windows/Mac/Linux 都能用

### 适合场景
- 前端开发（JavaScript/TypeScript）
- Python 开发
- 多语言项目（插件切换）
- 学生学习编程
- 产品经理体验开发工具（免费好上手）

### 不足
- 深度代码分析不如 JetBrains
- 大项目性能可能下降
- 需要自己装插件组装功能

---

## JetBrains（IntelliJ IDEA 等）

**专业 IDE 厂商，收费订阅**

### 核心特点
- **深度智能**：代码分析、重构能力业界最强
- **开箱即用**：功能内置，不用装一堆插件
- **收费**：个人版 $149/年，企业版更贵
- **语言专精**：每门语言有专用 IDE（IntelliJ for Java, PyCharm for Python）

### 适合场景
- Java/Kotlin 企业级开发
- 大型项目重构
- 需要深度代码分析的团队
- 专业开发团队预算充足

### 不足
- 重、启动慢
- 收费门槛
- 资源占用大（内存要求高）

---

## Cursor（AI 原生 IDE）

**2024 年爆火的 AI IDE，基于 VS Code 改造**

### 核心特点
- **AI 内置**：不是装插件，AI 是原生能力
- **代码库感知**：AI 能理解整个项目，不只是当前文件
- **Composer 模式**：一句话改多个文件
- **智能补全**：比传统补全更「预测意图」
- **聊天侧边栏**：随时问 AI 关于代码的问题

### 与 VS Code + Copilot 的区别
| 能力 | VS Code + Copilot | Cursor |
|------|------------------|--------|
| AI 集成 | 插件方式 | 原生深度集成 |
| AI 理解范围 | 当前文件/上下文 | 整个代码库 |
| 多文件编辑 | 手动逐个 | Composer 一键多文件 |
| 使用门槛 | 需配置 | 开箱即用 |

### 适合场景
- AI 协作开发
- 快速原型开发
- 产品经理用 AI 写需求/验证想法
- 代码新手（AI 帮写）

### 不足
- AI 功能收费（$20/月 Pro 版）
- 新产品，稳定性不如老牌 IDE
- 依赖 AI，可能有隐私顾虑

---

## 产品经理怎么选？

| 场景 | 推荐 |
|------|------|
| 想体验开发工具、零成本 | VS Code（免费） |
| 想用 AI 写需求、验证原型 | Cursor（AI 原生） |
| 公司有 JetBrains、想跟开发统一工具 | JetBrains |
| 不写代码，只是了解开发工作 | 不用装，知道 IDE 是什么就行 |
