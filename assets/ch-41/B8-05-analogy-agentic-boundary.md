## 类比一：AI 是厨师，人定菜单

### 类比场景

老李对小想说：

"AI 就像顶级厨师。你让它做菜，它刀工一流、火候精准。但菜单是谁定的？你定的。厨师不会替你决定'今天吃什么'。你想吃清淡还是辛辣，你想招待客人还是独自晚餐，这些品味决策都是你做的。厨师只负责把你的决策执行到位。"

### 对应概念映射

| 厨师场景 | Agentic Design 边界 |
|---------|---------------------|
| AI 是厨师 | AI 是执行者 |
| 人定菜单 | 人定品味决策 |
| 刀工一流火候精准 | AI 执行能力强 |
| 今天吃什么 | 产品方向、品牌形象 |
| 清淡还是辛辣 | 设计风格、用户体验取舍 |
| 招待客人还是独自晚餐 | 目标用户、市场定位 |

### 类比要点

- **AI 强项**：执行能力（刀工、火候）
- **人强项**：品味决策（菜单、口味）
- **边界清晰**：厨师不会替你定菜单

## 类比二：AI 是建筑师，人定风格

### 类比场景

老李说："你见过 AI 生成的建筑设计图吧？图很漂亮，结构合理，符合规范。但风格是什么？现代、古典、还是新中式？这个选择 AI 不会替你做。因为风格是品味，品味是人的事。AI 可以画十种风格的图，但你选哪一种，是你决定的。"

### 对应概念映射

| 建筑师场景 | Agentic Design 边界 |
|-----------|---------------------|
| AI 画设计图 | AI 生成设计方案 |
| 图漂亮结构合理 | 方案可行逻辑正确 |
| 符合规范 | 符合技术规范 |
| 风格选择 | 品味决策 |
| 现代/古典/新中式 | 设计风格取舍 |
| 人选哪一种 | 人定最终方向 |

### 类比要点

- **AI 提供**：十种风格的方案（可能性）
- **人决定**：选哪一种风格（品味）
- **边界**：选择比生成更关键

## 类比三：AI 是服装设计师，人定品牌

### 类比场景

小想让 AI 设计公司 logo，结果 AI 给出十个方案，小想却不知道选哪个。

老李问："你觉得哪个好？"

小想："都挺好看的，但不知道哪个合适。"

老李说："这就对了。AI 可以画出十个漂亮的 logo，但它不知道你的品牌是什么气质。是稳重？是活泼？是科技感？还是人文感？这些气质判断，AI 不懂。你先定义品牌气质，再让 AI 往那个方向生成，最后你来拍板。"

### 对应概念映射

| 服装设计师场景 | Agentic Design 边界 |
|---------------|---------------------|
| AI 设计十个 logo | AI 生成多个方案 |
| 都挺好看 | 方案质量高 |
| 不知道选哪个 | 品味判断困难 |
| 品牌气质 | 品牌定位 |
| 稳重/活泼/科技感/人文感 | 设计风格取舍 |
| 你先定义气质再让 AI 往那个方向生成 | 人定品味再 AI 执行 |

### 类比要点

- **AI 提供**：漂亮的设计方案
- **人提供**：品牌气质定义
- **关键步骤**：先定义品味，再让 AI 执行
- **最后一步**：人来拍板

## 类比四：AI 是助理，人定战略

### 类比场景

"AI 可以帮你写文案、做 PPT、分析数据。这些都是脑力活，AI 干得又快又好。但战略方向、市场判断、核心竞争力定位，这些 AI 不会替你做。为什么？因为战略需要品味判断，品味判断需要人的直觉、经验、价值观。AI 可以告诉你'市场数据是这样的'，但不能告诉你'我们应该往哪走'。"

### 对应概念映射

| 助理场景 | Agentic Design 边界 |
|---------|---------------------|
| AI 写文案做 PPT 分析数据 | AI 干脑力活 |
| 又快又好 | AI 执行能力强 |
| 战略方向市场判断 | 品味决策 |
| 核心竞争力定位 | 产品定位 |
| 人的直觉经验价值观 | 人定品味 |
| AI 告诉你市场数据是这样的 | AI 提供信息 |
| 不能告诉你应该往哪走 | 人定方向 |

### 类比要点

- **AI 干脑力活**：文案、PPT、数据分析
- **人定品味**：战略、方向、定位
- **信息 vs 决策**：AI 提供信息，人做决策
- **边界**：提供和决策是两回事

## 类比五：AI 是翻译官，人定表达

### 类比场景

"你让 AI 把你的产品介绍翻译成英文。AI 可以翻译得很流畅、很地道。但翻译的内容是什么？是你写的。你想表达什么语气？专业、亲切、还是幽默？是你决定的。AI 只是帮你把你的表达翻译成另一种语言，它不会替你决定'说什么'。"

### 对应概念映射

| 翻译官场景 | Agentic Design 边界 |
|-----------|---------------------|
| AI 翻译成英文 | AI 执行表达转换 |
| 流畅地道 | AI 执行能力强 |
| 翻译内容是你写的 | 内容是人定的 |
| 语气选择 | 品味决策 |
| 专业/亲切/幽默 | 表达风格取舍 |
| AI 把表达翻译成另一种语言 | AI 执行表达 |
| 不替你决定说什么 | 人定内容 |

### 类比要点

- **AI 执行**：语言转换、表达润色
- **人定内容**：说什么、语气是什么
- **边界**：表达和内容是两回事

## 品味决策清单

### AI 不擅长的事

| 类别 | 具体事项 | 为什么 |
|------|---------|--------|
| 品牌形象 | Logo 风格、品牌气质、视觉语言 | 需要审美品味 |
| 命名命名 | 产品名称、功能命名、术语选择 | 需要文化理解 |
| 核心插画 | 关键视觉、情感传递 | 需要艺术感知 |
| 新图标设计 | 图标风格、视觉系统 | 需要设计品味 |
| 战略方向 | 市场定位、核心竞争力 | 需要商业直觉 |
| 用户取舍 | 目标用户选择、体验权衡 | 需要价值观判断 |

### AI 擅长的事

| 类别 | 具体事项 | 为什么 |
|------|---------|--------|
| 文案执行 | 写具体文案、润色表达 | 语言能力强 |
| 方案生成 | 生成多个设计方案 | 创造力强 |
| 数据分析 | 分析市场数据、用户数据 | 计算能力强 |
| 代码编写 | 写代码、重构代码 | 编程能力强 |
| 文档撰写 | 写文档、整理资料 | 信息处理强 |
| 流程优化 | 优化流程、提升效率 | 逻辑能力强 |
