# Skill: chapter-draft

章节初稿写作能力

## 功能

根据验收标准（TDD）写作章节初稿，确保：
- 回答所有必须回答的问题
- 传达读者应获得的能力
- 纠正常见误解
- 使用准确术语

## 输入

- 验收标准文档（test_cases）
- 术语表引用
- 文风指南引用
- 素材需求（案例、类比、图示）

## 输出

- 章节初稿 Markdown 文件
- 包含完整10要素结构

## 执行流程

1. 解析验收标准，提取测试用例
2. 检查术语表，确认关键术语定义
3. 检查素材库，获取案例和类比
4. 按章节结构模板写作
5. 确保每个测试用例被覆盖
6. 执行自查清单

## 质量标准

- 所有测试用例 100% 覆盖
- 10要素结构完整
- 文风符合指南（短句、适度幽默、无空洞表达）
- 术语使用一致

## 依赖

- `term-lookup`：术语查询（运行时调用）
- 案例素材：由 [@案例师](mention://agent/2eb4c3b6-d91c-4372-9245-61769ab1032b) 在写作阶段前准备，存放于 `assets/cases/` 或 `manuscript/cases/`（非运行时依赖，通过工作流时序保证）

