# Skill: case-develop

案例开发与设计能力

## 功能

为章节开发案例素材：
- 正例案例（成功场景）
- 反例案例（失败场景）
- 场景构建
- 人物设定

## 输入

- 案例主题
- 案例类型（正例/反例）
- 涉及的概念

## 输出

- 完整案例内容
- 案例复盘要点
- 教学价值说明

## 执行流程

1. 分析案例需求
2. 设计案例人物（老李/小想）
3. 构建案例场景
4. 编写案例正文
5. 提炼复盘要点

## 案例人物设定

| 人物 | 特征 | 适用场景 |
|------|------|----------|
| 老李 | 40+，资深，经验丰富，不会写代码 | 复杂系统、判断力展示 |
| 小想 | 22岁，实习生，敢用AI，缺经验 | 新工具、踩坑展示 |

## 案例库目录

- 登录注册
- 权限管理
- 配置开关
- 审批流程
- 消息通知
- 搜索筛选
- 数据报表
- AI问答功能

## 质量标准

- 案例真实可信
- 有明确教学价值
- 人物行为符合设定

## Mention 语法

```
@case-writer 开发 [主题] 案例
@case-writer 为 [章节] 设计反例场景
```
