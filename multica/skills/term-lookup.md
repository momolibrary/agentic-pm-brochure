# Skill: term-lookup

术语表查询与维护能力

## 功能

管理书籍术语：
- 查询术语定义
- 检查术语使用一致性
- 新增术语定义
- 术语冲突检测

## 输入

- 术语名称
- 上下文

## 输出

- 术语定义
- 使用建议
- 相关术语

## 执行流程

1. 查询术语表
2. 返回定义和使用建议
3. 检测是否有冲突术语
4. 提供一致性建议

## 术语表结构

| 术语 | 定义 | 使用场景 | 别名 |
|------|------|----------|------|
| 接口 | 服务窗口比喻 | 系统交互 | API |
| 状态机 | 轨道切换器比喻 | 流程设计 | - |
| 灰度发布 | 小范围试吃比喻 | 发布策略 | - |

## 质量标准

- 定义准确
- 有类比解释
- 使用场景清晰

## Mention 语法

```
@reviewer 查询术语 [术语名]
@author-draft 确认术语 [术语名] 定义
```
