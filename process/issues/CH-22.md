## 验收标准

### 必须回答的问题

1. **IDE 是什么？VS Code、JetBrains、Cursor 有什么区别？**
   - 期望回答：IDE 是集成开发环境，开发写代码的核心工具。VS Code 免费插件多，JetBrains 专业收费，Cursor 是 AI 原生 IDE。产品经理也可以用 Cursor 做 AI 协作。
2. **Git 是什么？为什么产品也应该了解版本控制？**
   - 期望回答：Git 是版本控制系统，记录每次文件变更的历史。产品了解 Git 能更好地管理需求变更、追溯改动历史、与开发协作。
3. **SSH 是什么？运维说"SSH 到服务器看日志"是在做什么？**
   - 期望回答：SSH 是安全远程连接服务器的方式。"SSH 到服务器看日志"是远程登录服务器查看系统运行记录。

### 读者能做到的事

1. **能辨识 IDE/Git/SSH 等基础工具的角色和用途**
   - 验证方式：能说出每个工具解决什么问题
2. **能理解 VS Code/JetBrains/Cursor 的区别**
   - 验证方式：能根据场景推荐合适的 IDE
3. **能理解版本控制的基本概念**
   - 验证方式：能解释"分支""合并""冲突"的含义

### 必须纠正的误解

1. **误解："IDE 就是写代码的记事本"**
   - 正确理解：IDE 集成了语法高亮、调试、版本管理、终端、AI 辅助等能力，是开发的核心武器。
2. **误解："Git 是程序员的工具，跟产品无关"**
   - 正确理解：Git 的版本管理思维适用于任何需要追踪变更的场景，产品管理需求也可以用 Git。
